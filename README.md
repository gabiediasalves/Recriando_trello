# Recreate basic Trello using drag n' drop

![image](https://user-images.githubusercontent.com/95694730/172713957-543a623d-508f-48a7-a90d-2611190f47b2.png)

# Steps to use it
Install
```
yarn install
```
Start
```
yarn serve
```
Build
```
yarn build
```
